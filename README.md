# Backoffice Yala Numrung Group

ระบบจัดการ Backoffice สำหรับกลุ่มบริษัทยะลานำรุ่ง

## Features
- ระบบบุคคล (Employee Management)
- ระบบการลา (Leave Management)
- ระบบตั้งค่า (Settings)
- Premium Dark/Light Theme
- Role-based Access Control (Admin, HR, Supervisor, Employee)

## Tech Stack
- PHP 8+
- MySQL/MariaDB
- HTML5, CSS3, JavaScript
- jQuery, DataTables, SweetAlert2
- Google Sarabun Font

## Setup
1. Import database from `database/schema.sql`
2. Run seed data from `database/seed.sql`
3. Configure `includes/config.php` for database connection
4. Access via web browser

## Default Login
- Username: `admin`
- Password: `password`
